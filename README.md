
## Overview

Expense Tracker is a personal finance management application designed to help users monitor their income and expenses efficiently. The application provides an intuitive interface for recording transactions, organizing financial data, and gaining insights into spending habits.

## Features

* Add and manage income records
* Add and manage expense records
* Categorize transactions
* View financial summaries
* Track spending patterns
* Store and retrieve data using JSON
* User-friendly and responsive interface

## Technologies Used

### Frontend

* React.js
* HTML
* CSS
* JavaScript

### Backend

* Python

### Data Storage

* JSON

## Project Structure

* React.js handles the user interface and user interactions.
* Python processes data and manages application logic.
* JSON files are used for storing transaction data.

## Installation

### Clone the Repository

```bash
gh repo clone Lazib17/My_expensetracker
```

### Navigate to the Project Directory

```bash
cd expense-tracker
```

### Install Frontend Dependencies

```bash
npm install
```

### Start the React Application

```bash
npm start
```

### Run the Python Backend

```bash
python app.py
```

## Usage

1. Launch the application.
2. Add income sources and expenses.
3. Assign categories to transactions.
4. Monitor your financial activity through the dashboard.
5. Review spending trends and manage your budget effectively.
